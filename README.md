# Covid-19_Analysis
Bu veri, ülkelerin COVID-19 sürecindeki vaka yoğunluğu, ölüm oranları ve haftalık büyüme hızlarını karşılaştırmalı olarak analiz etmeye olanak tanımaktadır. Özellikle ölüm oranı ve haftalık artış metrikleri, sağlık sistemlerinin kriz yönetimi performansını değerlendirmek için kritik göstergelerdir.

Bu içerikte pandas,matplotlib ve seaborn kütüphaneleri kullanarak covid-19 verileri naaliz edilmiştir.
